# Product Photography

High-quality product photography is one of Tznami's most important marketing assets. This directory covers our lighting setup, camera settings, editing workflow, and shot list templates.

---

## Equipment

### Camera
- **Preferred**: Sony A7 III / A7C or Canon R6 (full-frame mirrorless for shallow DoF)
- **Minimum**: Any APS-C mirrorless with a 50mm equivalent lens
- **Lens**: 50mm f/1.8 or 90mm f/2.8 macro — tight compression, flattering perspective for keyboards
- **Tripod**: Essential for consistency and long exposures

### Lighting

**Option A — Softbox Setup (Studio)**
- 1× Key light: Large softbox (60cm × 90cm minimum), 45° above and to the left
- 1× Fill: White foam board reflector opposite key light, about 1 meter away
- Optional rim light: Small LED panel behind and slightly to the right for edge definition
- Color temperature: 5500K (daylight) for accurate color rendering

**Option B — Window Light (Natural)**
- North-facing window or overcast day (avoid direct sunlight — too harsh)
- White foam board fill on shadow side
- Best for lifestyle shots and editorial looks
- Shoot between 10am–2pm for consistent light

**Backdrop options**:
- **Off-white seamless paper** (2.7m wide, Colorama or equivalent): clean studio look
- **Tznami Navy felt or fabric**: dramatic dark background for colorway reveals
- **Minimal desk surface**: light wood or white desktop for lifestyle context
- **Avoid**: marble patterns, busy textures, or anything that competes with the keyboard

## Camera Settings

### Hero Product Shots (Studio)
| Setting | Value |
|---------|-------|
| ISO | 100–200 (base ISO for cleanest image) |
| Aperture | f/5.6–f/8 (for full keyboard in focus) |
| Shutter | 1/125s or use bulb with studio flash sync |
| White balance | Daylight / Custom (match your flash temperature) |
| Format | RAW (CR3 / ARW) — never JPEG for originals |
| Focus | Manual or single-point AF, refocus for each setup change |

### Detail Shots (Macro)
| Setting | Value |
|---------|-------|
| ISO | 100 |
| Aperture | f/2.8–f/4 (controlled shallow DoF) |
| Shutter | 1/200s or faster |
| Focus | Manual focus via focus peaking |
| Stabilization | Tripod + 2-second self-timer to eliminate shake |

### Video / Sound Test (Handheld)
| Setting | Value |
|---------|-------|
| ISO | 400–800 (low light tolerant) |
| Aperture | f/2.8 (cinematic DoF) |
| Shutter | 2× frame rate (1/50s at 25fps, 1/60s at 30fps) |
| Frame rate | 30fps for social, 60fps for slow-mo inserts |
| Picture Profile | S-Log2 or HLG for maximum editing latitude |

## Post-Processing Workflow

### Software
- **Adobe Lightroom Classic** (industry standard, excellent catalog management)
- **Darktable** (free, open source, Lightroom-comparable for RAW editing)
- **Capture One** (superior color science; recommended for tethered studio work)

### Editing Style
Tznami's editing style is refined and editorial — not over-processed:
1. Set correct white balance (sample a neutral gray area)
2. Exposure: slight exposure lift, protect highlights
3. Blacks: lift blacks slightly (crushed blacks feel harsh for product photography)
4. Texture/Clarity: +10–15 for product shots (brings out switch texture and case detail)
5. Color grading: cool midtones (slight cyan/blue), warm shadows (subtle amber)
6. Sharpening: Amount 50–70, Radius 0.8, Detail 30
7. Export: JPEG, quality 90%, sRGB, 2000px on longest edge for social; full-res TIFF for print/licensing
